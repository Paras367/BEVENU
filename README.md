## Overview

Bevenu is a production-grade, vanilla JavaScript e-commerce frontend built for premium fashion retail. It delivers a seamless, responsive shopping experience with modern UI patterns, persistent state management, and enterprise-ready architecture—all without external framework dependencies.

**Live Demo**: [https://bevenu.vercel.app](https://paras367.github.io/BEVENU/)  
**Admin Panel**: [https://bevenu.vercel.app/admin.html](https://paras367.github.io/BEVENU/admin.html) 

---

## Features

### Core Functionality
- Product browsing with advanced filtering (category, price, rating, sort)
- Shopping cart with quantity management and dynamic pricing
- Wishlist system with sidebar UI and persistence
- User authentication simulation (login/signup with localStorage)
- Admin panel for product CRUD operations
- Search with fuzzy matching and real-time suggestions
- Recently viewed products tracking
- Newsletter subscription with validation

### User Experience
- Responsive design (mobile, tablet, desktop)
- Dark/light mode toggle with system preference detection
- Smooth animations and micro-interactions (GSAP-powered)
- Skeleton loading states for perceived performance
- Toast notifications for user feedback
- Accessible keyboard navigation and ARIA labels
- Intersection Observer for scroll-triggered animations

### Technical Excellence
- Modular JavaScript architecture (IIFE pattern)
- CSS custom properties for theming and maintainability
- LocalStorage-based state management with prefix isolation
- Graceful CDN fallbacks for external libraries
- Debounced/throttled event handling for performance
- IntersectionObserver and ResizeObserver for efficient rendering
- Semantic HTML5 structure for SEO and accessibility
